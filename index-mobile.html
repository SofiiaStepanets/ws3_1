Цей сайт демонструє використання різних HTML елементів:

  • Списки (UL, OL, LI)
  • Зображення (IMG)
  • Посилання (A)
  • Карти зображень (MAP)

Списки (UL, OL, LI)

UL (Unordered List) — використовується для створення неупорядкованих списків, де пункти не мають конкретного порядку.

OL (Ordered List) — застосовується для створення упорядкованих списків, де пункти мають визначену нумерацію або індексацію.

LI (List Item) — елемент списку, що позначає окремий пункт у UL або OL.

Зображення (IMG)

IMG — елемент для вставки зображень на веб-сторінку. Він використовується для відображення графічних файлів і має атрибути для визначення джерела зображення, розмірів та альтернативного тексту.

Посилання (A)

A — тег для створення гіперпосилань. Використовується для того, щоб зв'язати одну веб-сторінку з іншою або з зовнішнім ресурсом. Посилання визначається через атрибут href, який вказує на URL-адресу.

Карти зображень (MAP)

MAP — тег, що дозволяє створювати інтерактивні карти зображень. Він визначає активні області на зображенні, які можуть виконувати певні дії при натисканні, наприклад, перенаправлення на інші сторінки. Для кожної активної області використовується тег AREA.
